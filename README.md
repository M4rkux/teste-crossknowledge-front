# teste-crossknowledge-front
Uma página com cadastro de Pessoas (nome, sobrenome e endereço).

Este projeto está responsável pelo frontend.

Para configurar a conexão com o backend, basta alterar a constante presente no arquivo script.js, alterando const URL_BASE = "http://crossknowledge.api/"; para o endereço do servidor de backend.

Este projeto foi feito utilizando apenas jquery e jquery.maskedinput, não foi adicionada nenhuma biblioteca de css ou framework de javascript.
